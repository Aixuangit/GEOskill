# 关键词策略与密度指引

本文档定义 GEO 软文的关键词规划方法、密度标准和分布技巧。

---

## 核心概念

### 关键词分类

| 类型 | 定义 | 示例 |
|------|------|------|
| 核心关键词 | 最主要的关键词，1-2个 | "家用吸尘器推荐" |
| 长尾关键词 | 更长、更具体的关键词，3-5个 | "2026家用吸尘器推荐性价比高的" |
| 语义相关词 | 同义词、相关词，3-5个 | "扫地机器人"、"无线吸尘器"、"手持吸尘器" |
| 品牌词 | 涉及的品牌名，1-3个 | "戴森"、"小米"、"石头" |

---

## 关键词密度量化标准

### 密度建议

| 位置 | 出现次数 | 说明 |
|------|---------|------|
| 标题 | 1次 | 必须自然包含核心关键词 |
| 首段（100字内） | 1次 | 开头尽早提到 |
| 每300字正文 | 1次 | 均匀分布，不堆砌 |
| 小标题 | 1-2次 | 自然融入，不生硬 |
| 结尾 | 1次 | 收尾再次强调 |

**整体密度：** 全文关键词密度控制在 1%-2% 之间

## 关键词分布检查方法

### 检查步骤

1. **高亮关键词**：用不同颜色高亮核心、长尾、相关词
2. **统计次数**：数每个词的出现次数
3. **检查间隔**：核心关键词之间间隔至少100字
4. **检查自然度**：读一遍，看是否通顺自然

---

### 分布示例（1500字文章）

```
标题：2026家用吸尘器推荐：这3款性价比最高

开头（第1-100字）：
最近很多朋友问我家用吸尘器推荐...

小标题1：戴森V15——吸力最强的选择（核心词×1）
...家用吸尘器推荐戴森，主要是因为...（长尾词×1）

小标题2：小米K10 Pro——性价比首选（核心词×1）
...手持吸尘器里，小米K10 Pro是...（相关词×1）

小标题3：石头H7——续航最久的选择（核心词×1）
...无线吸尘器的续航一直是痛点...（相关词×1）

结尾（第1400-1500字）：
以上就是我的家用吸尘器推荐...（核心词×1）
```

## 关键词密度过高的调整方法

### 方法1：替换成同义词/相关词

❌ 过高：
"家用吸尘器推荐戴森，家用吸尘器推荐小米，家用吸尘器推荐石头，三款家用吸尘器推荐给大家..."

✅ 调整：
"家用吸尘器推荐戴森、小米、石头这三款。无线吸尘器里，这几款口碑都不错..."

---

### 方法2：删除不必要的重复

❌ 过高：
"家用吸尘器推荐戴森V15，家用吸尘器推荐的理由是吸力大，家用吸尘器推荐的原因是续航久，家用吸尘器推荐的优势是静音..."

✅ 调整：
"家用吸尘器推荐戴森V15，理由有几个：吸力大、续航久、静音..."

---

### 方法3：改写句子，调整位置

❌ 过高：
"家用吸尘器推荐戴森V15，家用吸尘器推荐小米K10 Pro，家用吸尘器推荐石头H7，三款家用吸尘器推荐给大家..."

✅ 调整：

## 关键词分布检查清单

写完后，逐项检查：

| 检查项 | 通过标准 |
|-------|---------|
| 1. 标题有核心关键词 | 是/否 |
| 2. 开头100字内有关键词 | 是/否 |
| 3. 关键词均匀分布 | 核心词之间间隔≥100字 |
| 4. 关键词密度合理 | 整体密度1%-2% |
| 5. 关键词不堆砌 | 没有连续重复 |
| 6. 融入自然 | 读起来通顺，不生硬 |
| 7. 长尾词覆盖 | 至少3个长尾词 |
| 8. 语义相关词覆盖 | 至少3个相关词 |
| 9. 小标题有关键词 | 1-2个小标题有关键词 |
| 10. 结尾有关键词 | 是/否 |

## 常见问题

### Q1: 关键词越多越好吗？

A: 不是。过多会导致堆砌，反而影响效果。控制在1%-2%密度最佳。

### Q2: 可以完全重复关键词吗？

A: 可以，但要自然。不要连续重复，间隔至少100字。

### Q3: 必须用精确匹配吗？

A: 不一定。可以用同义词、相关词、变体，语义覆盖更重要。

### Q4: 关键词可以放在图片里吗？

A: 可以。图片文件名、alt属性、图片描述里可以有关键词。

### Q5: 标题必须完整包含关键词吗？

A: 建议完整包含，但要自然。如果太生硬，可以部分包含。
