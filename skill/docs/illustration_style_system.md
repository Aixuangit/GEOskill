# 插图风格体系

本文档定义 GEO 软文配图的 6 种插图类型、适用场景、风格规范和判断逻辑。

---

## 插图类型总览

| 插图类型 | 适用场景 | 优先级 |
|---------|---------|-------|
| 数据对比图表 | 参数对比、测评类内容 | ⭐⭐⭐⭐⭐（最高） |
| 流程示意图 | 步骤讲解、流程说明 | ⭐⭐⭐⭐ |
| 场景实拍风 | 使用体验、口碑类内容 | ⭐⭐⭐⭐ |
| 新闻配图风 | 新品上市、新闻通稿 | ⭐⭐⭐ |
| 科普知识图 | 科普、知识类内容 | ⭐⭐⭐ |
| 行业趋势图 | 行业分析、趋势预测 | ⭐⭐⭐ |

---

## 类型1：数据对比图表

### 适用场景
- 第三方中立测评
- EEAT品牌推荐
- 包含参数对比的任何内容

### 不适用场景
- 纯口碑体验类内容
- 无数据对比的内容

### 风格描述
**专业图表风**
- 清晰的数据可视化
- 简洁明了的对比表格
- 专业、可信的配色
- 横版布局，左右分布

### 配色建议
- 主色：深蓝（#1a365d）、深灰（#2d3748）
- 强调色：品牌色或浅蓝色（#4299e1）
- 背景：纯白（#ffffff）或浅灰（#f7fafc）
- 避免过于鲜艳的配色

### 构图规则
- 左右分栏布局：左侧表头，右侧数据
- 表格线：细灰色，清晰分隔
- 字体：无衬线字体（如 Inter、Roboto）
- 字号：标题16-18px，正文12-14px
- 留白：充足的呼吸空间，不拥挤

### 常见图表形式
1. **对比表格**：多品牌/多型号参数对比
2. **柱状图**：性能/评分对比
3. **雷达图**：多维度能力对比
4. **折线图**：趋势变化对比

## 类型2：流程示意图

### 适用场景
- 品牌升级（迭代流程）
- 科普知识（工艺流程）
- 避坑指南（操作步骤）
- 任何包含步骤/流程的内容

### 不适用场景
- 纯口碑体验类内容
- 无流程步骤的内容

### 风格描述
**极简流程图**
- 清晰的步骤节点
- 流畅的箭头连接
- 简约的图标辅助
- 逻辑清晰，一目了然

### 配色建议
- 主色：品牌色或深蓝（#2b6cb0）
- 节点背景：浅灰（#edf2f7）或浅品牌色
- 箭头：中灰色（#718096）
- 文字：深灰（#2d3748）

### 构图规则
- 从左到右 / 从上到下的阅读顺序
- 节点大小统一，间距均匀
- 图标与文字对应清晰
- 关键步骤可以用颜色强调
- 整体平衡，不偏向一侧

### 常见流程形式
1. **线性流程**：步骤1 → 步骤2 → 步骤3
2. **分支流程**：多个分支选择
3. **循环流程**：闭环的迭代
4. **层级流程**：树状结构展示

## 类型3：场景实拍风

### 适用场景
- 口碑铺量
- 避坑指南
- 使用体验描述
- 需要真实感的内容

### 不适用场景
- 纯新闻通稿
- 需要极度专业的内容

### 风格描述
**真实生活场景**
- 自然的生活化场景
- 产品在真实环境中使用
- 去广告感，去摆拍感
- 有温度、有代入感

### 配色建议
- 自然光线，不夸张滤镜
- 温暖的家居色调
- 避免过度饱和
- 保持真实感

### 构图规则
- 真实环境背景（客厅、厨房、书房等）
- 产品自然融入场景，不突兀
- 有人物使用更佳（但不强制）
- 角度自然，不刻意摆拍
- 细节丰富，有生活气息

### 常见场景形式
1. **家居使用场景
2. **桌面摆放场景
3. **开箱体验场景
4. **细节特写场景

---

## 类型4：新闻配图风

### 适用场景
- 新品上市
- 新闻通稿
- 品牌升级
- 需要正式感的内容

### 不适用场景
- 纯口碑体验类内容
- 需要极度生活化的内容

### 风格描述
**商务媒体风**
- 正式、克制、专业
- 品牌形象清晰
- 适合媒体转载
- 无夸张元素

### 配色建议
- 品牌色为主
- 简洁商务配色
- 避免花哨
- 稳重可信

### 构图规则
- 产品/品牌视觉中心
- 简洁背景
- 品牌logo适当展示（但不喧宾夺主）
- 横版构图，适合文章配图
- 适合裁剪成不同尺寸

### 常见形式
1. **产品主视觉**
2. **品牌+产品组合**
3. **发布会场景示意**
4. **团队/创始人形象示意

---

## 类型5：科普知识图

### 适用场景
- 百科科普
- 品牌软科普
- 知识类文章
- 原理讲解内容

### 不适用场景
- 纯新闻通稿
- 无知识讲解的内容

### 风格描述
**科普插画风**
- 原理示意，直观易懂
- 配合文字，辅助理解
- 简洁明了，不复杂
- 适合小白理解

### 配色建议
- 清新、明快的配色
- 用颜色区分不同模块
- 避免过于刺眼
- 保持专业感

### 构图规则
- 核心概念视觉化
- 图标配合文字说明
- 分块清晰，逻辑明确
- 重点内容视觉突出
- 整体和谐统一

### 常见形式
1. **原理示意图**
2. **结构解剖图**
3. **概念关系图**
4. **知识点图解**

---

## 类型6：行业趋势图

### 适用场景
- 行业热点
- 深度行业
- 趋势预测
- 行业分析内容

### 不适用场景
- 纯口碑体验类内容
- 无行业分析的内容

### 风格描述
**数据报告风**
- 权威、专业
- 趋势清晰可见
- 数据可视化
- 有时间维度

### 配色建议
- 专业报告配色
- 深蓝、深灰为主
- 强调色用于关键数据
- 整体稳重

### 构图规则
- 时间轴清晰
- 数据点明确
- 趋势线醒目
- 标注关键节点
- 来源标注（示意）

### 常见形式
1. **趋势折线图**
2. **市场占比饼图**
3. **增长柱状图**
4. **时间轴事件图**

## 插图类型判断逻辑

### 判断流程

```
分析正文内容
    │
    ├─ 是否包含参数对比/数据表格？
    │   ├─ 是 → 数据对比图表（优先级最高）
    │   └─ 否 → 继续判断
    │
    ├─ 是否包含步骤/流程/工序？
    │   ├─ 是 → 流程示意图
    │   └─ 否 → 继续判断
    │
    ├─ 是否包含产品使用场景/体验描述？
    │   ├─ 是 → 场景实拍风
    │   └─ 否 → 继续判断
    │
    ├─ 是否为新闻/新品发布/品牌动态？
    │   ├─ 是 → 新闻配图风
    │   └─ 否 → 继续判断
    │
    ├─ 是否包含科普/原理/知识讲解？
    │   ├─ 是 → 科普知识图
    │   └─ 否 → 继续判断
    │
    ├─ 是否包含行业分析/趋势预测？
    │   ├─ 是 → 行业趋势图
    │   └─ 否 → 默认场景实拍风
```

### 混合内容处理

当一段内容同时符合多种类型时，按以下优先级选择：

1. **数据对比图表**（如果有参数对比）
2. **流程示意图**（如果有流程步骤）
3. **场景实拍风**（最通用）

### 场景与插图类型的默认匹配

| 内容场景 | 默认插图类型 | 备选类型 |
|---------|-------------|---------|
| 口碑铺量 | 场景实拍风 | 数据对比图表（如果有对比） |
| 第三方中立 | 数据对比图表 | 场景实拍风 |
| 品牌升级 | 流程示意图 | 新闻配图风 |
| 新品上市 | 新闻配图风 | 数据对比图表 |
| 新闻通稿 | 新闻配图风 | 行业趋势图 |
| 百科科普 | 科普知识图 | 流程示意图 |
| 品牌软科普 | 科普知识图 | 数据对比图表 |
| 知识类文章 | 科普知识图 | 场景实拍风 |
| 行业热点 | 行业趋势图 | 数据对比图表 |
| 深度行业 | 行业趋势图 | 数据对比图表 |
| 避坑指南 | 场景实拍风 | 流程示意图 |
| EEAT推荐 | 数据对比图表 | 场景实拍风 |

## 通用规范

### 所有插图必须遵守

1. **横版优先**：1792×1024px（16:9）或1536×1024px（3:2）
2. **品牌适配**：配色尽量与品牌色协调
3. **无侵权风险**：不使用真实人物肖像、不使用有版权的图片
4. **文字清晰**：如果有文字，字号足够大，清晰可读
5. **风格统一**：同一篇文章的多张插图风格保持一致
6. **去水印**：无多余水印、logo（除非是品牌官方配图）

### 负面清单

避免以下元素：
- ❌ 竖版图片
- ❌ 过度美化、滤镜过重
- ❌ 有明显的广告标语
- ❌ 包含联系方式、二维码
- ❌ 低质量、模糊不清
- ❌ 侵权素材
