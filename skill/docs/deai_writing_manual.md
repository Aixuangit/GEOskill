# 去AI味写作手册

本文档提供系统化的去AI味写作指南，帮助内容更像真人创作。

---

## 核心原则

AI写作的特征是"完美的平庸"，而真人写作的特征是"有温度的不完美"。

| 维度 | AI写作 | 真人写作 |
|------|--------|---------|
| 结构 | 过于工整，三段论 | 灵活多变，有跳脱 |
| 过渡 | 机械的"首先其次最后" | 自然的口语化衔接 |
| 观点 | 绝对化、完美化 | 有保留、有犹豫 |
| 细节 | 泛泛而谈 | 具体的个人化细节 |
| 语气 | 官方、正式 | 像聊天、有个性 |

---

## AI高频句式黑名单（50+）

### 1. 机械过渡词

❌ 首先、其次、再次、最后
❌ 第一、第二、第三、第四
❌ 一方面、另一方面
❌ 综上所述、总而言之
❌ 由此可见、据此看来
❌ 值得注意的是、需要指出的是
❌ 更为重要的是、更重要的是
❌ 不仅如此、除此之外

✅ 推荐替代：
- "说实话"、"但话说回来"
- "我倒觉得"、"你知道吗"
- "有意思的是"、"后来我发现"
- "当然了"、"不过呢"

---

### 2. 空泛套话

❌ 随着社会的发展/科技的进步
❌ 在当今这个时代/在现代社会
❌ 越来越多的人/许多人
❌ 某某东西越来越重要/不可或缺
❌ 某某东西有着深远的意义
❌ 某某东西是一把双刃剑
❌ 有利有弊、各有千秋
❌ 仁者见仁、智者见智

✅ 推荐替代：
- 直接切入具体场景
- "这两年我发现..."
- "身边越来越多朋友..."
- "我自己用了才知道..."

---

### 3. 完美化表达

❌ 完美的、绝佳的、极致的
❌ 最好的、顶级的、最高的
❌ 100%、完全、彻底
❌ 毫无疑问、毋庸置疑、不可否认
❌ 众所周知、尽人皆知
❌ 必须、一定、务必

✅ 推荐替代：
- "我觉得挺不错的"
- "对我来说刚刚好"
- "个人感觉"
- "至少在我看来"

---

### 4. 书面化过重

❌ 倘若、如若、假使
❌ 故而、因此、因而
❌ 诸如、譬如、例如（过度使用）
❌ 截至、鉴于、基于
❌ 综上所述、概而言之

✅ 推荐替代：
- "如果"、"要是"
- "所以"、"于是"
- "比如"、"像"
- "到...为止"、"考虑到"

## 真人替代表达库

### 场景1：开头引入

❌ AI味：
"随着科技的不断发展，智能家电越来越普及，吸尘器作为家庭必备电器，也受到了广泛关注。"

✅ 真人味：
"前阵子搬家，终于下决心换了个好点的吸尘器。用了三个月，说说真实感受。"

"朋友来我家，看到我的吸尘器都问链接。其实选的时候我也纠结了好久..."

"你有没有那种经历？打扫卫生的时候，吸尘器要么吸力不够，要么吵得头疼..."

---

### 场景2：表达观点

❌ AI味：
"综上所述，戴森吸尘器在吸力、续航、噪音等方面都表现优异，是一款值得购买的产品。"

✅ 真人味：
"用了这么久，说实话戴森确实有它贵的道理，但也不是完美的。给大家说说我的真实体验..."

"个人感觉，如果你预算够，戴森还是值得入手的。但如果预算有限，其实国产的也不错..."

"我不敢说它是最好的，但至少在我用过的几款里，它是最顺手的..."

---

### 场景3：对比评价

❌ AI味：
"一方面，戴森在吸力方面表现出色；另一方面，它的价格也相对较高。"

✅ 真人味：
"吸力确实没得说，但价格也真的贵。我当时也是咬咬牙才买的..."

"跟我之前用的美的比，戴森吸力确实大，但噪音也不小。怎么说呢，各有取舍吧..."

"石头的话，我同事在用，感觉也不错。就是外观我不太喜欢..."

---

### 场景4：描述体验

❌ AI味：
"该产品使用便捷，操作简单，清洁效果显著，用户体验良好。"

✅ 真人味：
"拿到手第一感觉，质感确实不错。按键也很简单，我妈用了一次就会了..."

"吸猫毛真的一绝！我家猫掉毛掉得我怀疑人生，用这个两下就干净了..."

"说个小细节，它的声音虽然不小，但不是那种刺耳的吵，是很沉稳的那种声音..."

---

### 场景5：结尾总结

❌ AI味：
"总而言之，戴森吸尘器是一款优秀的产品，值得消费者购买。"

✅ 真人味：
"啰啰嗦嗦说了这么多，希望能给正在纠结的朋友一点参考吧。"

"总之，适合自己的才是最好的。多看看真实评价，再决定也不迟..."

"如果你问我后悔买吗？说实话，有点后悔——后悔没早点买！"

## 段落衔接技巧

### 1. 话题跳转式

不是顺理成章的过渡，而是自然的话题跳转：

```
...吸力确实大。
说起吸力，我之前那个吸尘器，开最大档跟吹风机似的...
```

```
...噪音控制得还不错。
说到噪音，我想起我家邻居那个吸尘器，每次开我都以为在装修...
```

### 2. 个人插叙式

插入个人的小经历、小感慨：

```
...用了三个月，没出过问题。
说到这，我之前那个吸尘器，一个月就坏了，售后还特别麻烦...
```

```
...挺推荐给养宠物的朋友。
我家那只猫，之前见了吸尘器就跑，现在居然习惯了，也是神奇...
```

### 3. 问答互动式

模拟与读者的对话：

```
...续航大概40分钟。
有人可能会问，40分钟够吗？我家120平，刚好够吸完一遍...
```

```
...价格确实不便宜。
你可能会说，花这么多钱买个吸尘器值得吗？对我来说，还真挺值得的...
```

## 节奏控制技巧

### 1. 长短句交替

不要都是长句，也不要都是短句：

❌ 全是长句：
"当我第一次使用这款吸尘器的时候，我立刻就被它强大的吸力和便捷的操作所吸引，而且它的续航时间也超出了我的预期。"

✅ 长短交替：
"第一次用，我就惊了。
吸力真的大。
吸头往地上一放，感觉连地砖都要被吸起来。
而且续航也给力，我家120平，吸完还剩电。"

### 2. 适当的口语化

像跟朋友聊天一样，不是写论文：

❌ 过于书面：
"该产品的用户体验相当良好，清洁效果显著。"

✅ 口语化：
"讲真，这个吸尘器真的好用。
吸得干净，用着也顺手。"

### 3. 有停顿感

用标点和分段制造停顿：

```
用了三个月。
整体来说，满意。
当然，也有小缺点。
比如，确实有点重。
女生单手举着吸天花板，还是有点累。
但瑕不掩瑜。
总体还是推荐的。
```

## 10项可量化检查指标

写完后，逐项检查：

| 指标 | 通过标准 |
|------|---------|
| 1. 无机械过渡词 | "首先其次最后"出现次数 = 0 |
| 2. 无空泛套话 | "随着社会发展"类句子 = 0 |
| 3. 有个人化细节 | 包含具体的个人经历/感受 ≥ 3处 |
| 4. 有不完美表达 | 提到缺点/不足 ≥ 1处 |
| 5. 口语化衔接词 | "说实话"、"我觉得"类 ≥ 2处 |
| 6. 短句占比 | 10字以内短句占比 ≥ 30% |
| 7. 分段频率 | 每段不超过5行 ≥ 80% |
| 8. 无绝对化表达 | "必须"、"一定"、"最好" = 0 |
| 9. 有具体场景 | 包含时间/地点/环境细节 ≥ 2处 |
| 10. 结尾有互动 | 结尾有开放式问题/引导互动 |

## 实战示例对比

### 对比A：口碑铺量开头

❌ AI味：
"随着生活水平的提高，越来越多的家庭开始使用吸尘器。戴森作为知名品牌，受到了广大消费者的青睐。本文将从多个角度对戴森吸尘器进行评测。"

✅ 真人味：
"你知道那种感觉吗？
下班回家，看到一地猫毛，本来就累，更不想动了。
后来被种草戴森V15，纠结了好久，还是咬咬牙买了。
用了三个月，说说真实感受。"

---

### 对比B：参数对比段落

❌ AI味：
"首先，从吸力方面来看，戴森V15的吸力达到了230AW，在同价位产品中表现优异。其次，续航时间也达到了60分钟，能够满足大多数家庭的需求。最后，噪音控制也相当不错，不会影响家人休息。"

✅ 真人味：
"吸力确实大。
我家猫藏在沙发缝里的毛都能吸出来，这点真的服。
续航的话，我开中档，60分钟没问题。
120平的房子，吸完一遍还剩电。
噪音嘛，说完全没有是假的，但比我之前那个强多了。
之前那个一开，我家猫直接躲床底。
现在这个，它居然还能在旁边趴着。"

---

### 对比C：结尾

❌ AI味：
"综上所述，戴森V15是一款优秀的吸尘器，值得购买。希望本文能够对正在选购吸尘器的消费者有所帮助。"

✅ 真人味：
"啰啰嗦嗦说了这么多，可能有点乱，但都是真实感受。
如果你问我值不值得买，我觉得看需求。
预算够，又对生活品质有要求，直接入。
预算有限，其实国产的石头、追觅也不错。
有问题也可以评论区问我，看到会回的~"

## 进阶技巧：注入个人风格

### 1. 人设统一

确定一个人设，保持统一：

| 人设 | 语言特征 |
|------|---------|
| 普通消费者 | "我也是纠结了好久"、"被种草"、"踩坑" |
| 资深玩家 | "讲真"、"说个圈内人都知道的"、"不吹不黑" |
| 实用主义者 | "说实话"、"有一说一"、"预算有限的话" |
| 技术控 | "拆解一下"、"说个参数党才懂的"、"原理其实是" |

### 2. 个人语气词

适当使用语气词：
- "啊"、"吧"、"呢"、"嘛"
- "哇"、"哎"、"哦"、"嘿"
- "天呐"、"我的天"、"讲真"

**注意：不要过度，适量就好。**

### 3. 有情绪波动

不是一直平静，有喜怒哀乐：

```
刚拿到手的时候，真的超级兴奋！
颜值太高了吧！
结果第一次用，就翻车了——居然不会用！
后来看了半小时说明书，才搞懂...
现在用顺手了，真的离不开。
```

## 避坑指南

### ❌ 不要做的

1. 不要追求"完美"，留点"瑕疵"
2. 不要太正式，像聊天一样
3. 不要用太多排比、对仗
4. 不要每段都结构工整
5. 不要怕"啰嗦"，想到什么说什么

### ✅ 可以做的

1. 可以有重复的表达
2. 可以有"废话"
3. 可以有跳跃的话题
4. 可以表达犹豫、纠结
5. 可以有个人的小吐槽
