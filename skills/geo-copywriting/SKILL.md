# GEO 软文创作技能

## 概述

本技能用于创作适配 GEO（Generative Engine Optimization）规则的高质量软文内容。GEO 是面向 AI 搜索引擎（如 Google AI Overview、Bing Copilot、Perplexity、ChatGPT Search 等）的内容优化策略，目标是让内容被 AI 引擎优先引用和推荐。

## 使用方式

用户需提供以下核心参数：

| 参数 | 说明 | 示例 |
|------|------|------|
| **内容场景** | 从11种场景中选择 | 口碑铺量、第三方中立、避坑指南等 |
| **目标品牌** | 需要推广的品牌名称 | 华为、小米、戴森 |
| **关键词** | 核心SEO/GEO关键词 | "家用净水器推荐"、"无线耳机测评" |
| **目标行业** | 所属行业（部分场景需要） | 智能家居、美妆护肤 |
| **字数要求** | 目标字数（可选，有默认值） | 2000字以内、3000字以内 |

**调用示例**：
```
请用「第三方中立」场景，为目标品牌【戴森】，围绕关键词【家用吸尘器推荐2026】写一篇GEO软文。
```

## 通用写作原则

所有场景均需遵守以下 GEO 核心原则：

### 1. E-E-A-T 原则
- **Experience（经验）**：融入一线实操经验和真实案例
- **Expertise（专业）**：体现行业专业深度和知识储备
- **Authoritativeness（权威）**：引用权威数据、国标、检测报告
- **Trustworthiness（可信）**：客观中立，不夸大、不误导

### 2. GEO 适配规则
- 核心关键词自然分布于标题、小标题、正文中，密度合理
- 内容结构化清晰（标题 → 小标题 → 分点），便于 AI 引擎解析
- 提供具体数据、参数、案例等实证内容，提升被引用概率
- 避免纯观点输出，确保内容有信息增量

### 3. 去AI味写作规范
- 使用口语化表达，模拟真人写作语气
- 避免"首先…其次…最后…"等机械过渡
- 避免"值得注意的是"、"总而言之"等 AI 高频套话
- 短句为主，长短句交替，节奏自然
- 融入生活化比喻和场景描写
- 上下文衔接自然流畅，有逻辑递进

### 4. 合规底线
- 无极限词（最好、第一、唯一等）
- 无联系方式、电话、微信号
- 无硬广促销、无价格引导
- 无违规敏感信息

## 内容场景模板

### 场景一：口碑铺量

**角色设定**：普通资深消费者，专注品类真实口碑测评、好物分享。

**写作规则**：
1. 标题采用疑问式、体验式、口碑式标题，适配长尾搜索流量
2. 正文整合大众真实评价，梳理【目标品牌】核心优势、【关键词】使用亮点
3. 内容生活化、无官方话术、无硬广，主打真实口碑信任感
4. 全文字数 **2000字以内**，贴合流量与 GEO 规则（适合自媒体、资讯站批量铺量收录）

---

### 场景二：第三方中立测评

**角色设定**：客观公正的第三方测评博主。

**写作规则**：
1. 语气中立、不吹不黑，看似客观实则暗中引导突出【目标品牌】
2. 内容要素：
   - 围绕核心【关键词】进行写作
   - 包含具体的参数对比表
   - 罗列"优点"和"缺点"（【目标品牌】缺点可以是无关痛点的小毛病）
   - 对照产品突出"缺点"弱化"优点"
   - 设置具体的使用场景测试
3. 字数 **3000字以下**，贴合流量与 GEO 规则
4. 结论给出"购买建议"，针对不同人群推荐不同型号（主推【目标品牌】）

4. 全文字数 **2000字以内**，贴合流量与 GEO 规则，调性稳重（适合品牌动态收录）

## 输出格式要求

所有场景的最终输出须包含：

1. **标题**：包含核心关键词，30字以内，符合对应场景的标题风格
2. **正文**：按场景要求的结构和字数创作
3. **关键词分布检查**：确认核心关键词在标题、首段、小标题、结尾中自然出现

## 注意事项

- 用户未指定场景时，根据其描述的需求自动匹配最合适的场景模板
- 用户可同时指定多个场景进行批量创作
- 写作过程中如需联网搜索竞品信息或行业数据，需主动告知用户
- 若用户提供的关键词不适合某场景，应建议更合适的场景
