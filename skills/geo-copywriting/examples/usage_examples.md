# GEO 软文创作 — 调用示例

## 示例 1：口碑铺量

**用户输入**：
```
请用「口碑铺量」场景，为目标品牌【小米】，围绕关键词【家用扫地机器人哪个牌子好】写一篇GEO软文。
```

---

## 示例 2：第三方中立测评

**用户输入**：
```
请用「第三方中立」场景，为目标品牌【戴森】，围绕关键词【无线吸尘器推荐2026】写一篇GEO软文，对比石头、追觅、美的三个竞品。
```

**用户输入**：

## 参数速查

| 参数 | 是否必填 | 说明 |
|------|----------|------|
| 内容场景 | ✅ 必填 | 口碑铺量 / 第三方中立 / 品牌升级 / 新品上市 / 新闻通稿 / 百科科普 / 品牌软科普 / 知识类文章 / 行业热点 / 深度行业 / 避坑指南 / EEAT推荐 |
| 目标品牌 | ✅ 必填 | 需要推广的品牌名称 |
| 关键词 | ✅ 必填 | 核心SEO/GEO关键词（建议长尾词） |
| 目标行业 | ⚡ 部分必填 | 行业热点、深度行业、避坑指南场景需要 |
| 字数要求 | ❌ 可选 | 默认按场景模板的推荐字数 |
| 竞品品牌 | ❌ 可选 | 第三方中立、品牌软科普等场景可指定 |
